# 🧾 Sistema POS — Refaccionaria "Terry"

Sistema de punto de venta (POS) de escritorio desarrollado en **C# y .NET Framework**, con **SQL Server** como motor de base de datos, creado para la gestión de ventas y control de inventario de una refaccionaria.

## 📌 Descripción

Proyecto desarrollado como parte de la materia de Programación II, implementado y utilizado en tiempo real en la operación diaria de una refaccionaria local. El sistema permitía registrar ventas, gestionar el inventario de productos y consultar información del negocio, reemplazando procesos que antes se llevaban de forma manual.

> El negocio para el que se desarrolló este sistema cerró operaciones durante la pandemia debido a la caída de ventas, pero el sistema estuvo en uso real durante su operación.

## 🛠️ Tecnologías utilizadas

- **Lenguaje:** C#
- **Framework:** .NET Framework
- **Base de datos:** SQL Server
  - Queries
  - Stored Procedures
  - Triggers
  - Views
  - Jobs

## ⚙️ Funcionalidades principales

- Registro y gestión de ventas.
- Control de inventario de productos.
- Consultas y reportes mediante procedimientos almacenados.
- Automatización de procesos a nivel de base de datos (triggers y jobs).

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/eduardo-marmolejo/ProgramacionII.git
   ```
2. Restaurar la base de datos en SQL Server usando el script `REFACCIONARIA-TBD.sql`.
3. Abrir el proyecto en Visual Studio.
4. Configurar la cadena de conexión a la base de datos.
5. Ejecutar la aplicación.
